# DRL-Solutions
Reinforcement Learning Solutions 2d Edition 2018

# Setup

The code works with python3 and virtualenvs.

```
$ pip install virtualenv
$ cd <work_directory>
$ virtualenv -p python3 .env
$ source .env/bin/activate
$ pip install -r requirements.txt
$ deactivate 
```
# Run code 

```
$ cd <work_directory>
$ source .env/bin/activate
...

run code

....
$ deactivate 
```
